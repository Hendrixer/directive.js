# directive.js

Directive.js brings the power of directives from Angular into VanillaJS.

## Building it

Run:

- `sudo npm install -g gulp gulp-cli`
- `npm install`
- `gulp package`

The files will be created inside of `dist/directive.js`.

## Testing it

Run:

- `sudo npm install -g gulp gulp-cli`
- `npm install`
- `gulp test`
